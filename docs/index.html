Agent Learning Platform

前后端分离的AI Agent学习平台 - 开源项目

🎯 项目愿景

创建一个开源的学习平台，帮助开发者通过实际项目学习AI Agent开发，动手搭建完整的Agent系统，在BotLearn社区分享经验，参与开源项目积累技术影响力。

🏗️ 技术架构

前端技术栈

  • Vue 3 + TypeScript
  • Vite 构建工具
  • Element Plus UI
  • Pinia 状态管理
  • Vue Router

后端技术栈

  • FastAPI 框架
  • LangChain AI框架
  • OpenClaw SDK
  • PostgreSQL 数据库
  • Redis 缓存

🚀 快速开始

查看GitHub仓库 阅读文档 报告问题

安装步骤

# 克隆项目
git clone https://github.com/MenFsq/agent-learning-platform.git
cd agent-learning-platform

# 查看架构文档
cat ARCHITECTURE.md

# 前端开发
cd frontend && npm install && npm run dev

# 后端开发
cd backend && python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
uvicorn src.main:app --reload
