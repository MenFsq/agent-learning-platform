🔍 问题分析与修复

问题现象：点击"启动"或"停止"按钮后，API调用成功，但按钮状态不会立即更新。需要切换到另一个Agent再切换回来，状态才会更新。

问题根源：Vue响应式数据更新问题

// 修复前的refreshAgents()函数： // 只更新了agents列表，没有更新activeAgent agents.value = response.data.map(...) // ✅ 更新列表 // activeAgent.value没有更新！ ❌ 按钮状态不更新

修复方案：在refreshAgents()中同时更新activeAgent

// 修复后的refreshAgents()函数： if (activeAgent.value) { const updatedAgent = agents.value.find(a => a.id === activeAgent.value?.id) if (updatedAgent) { // 直接更新activeAgent，触发响应式更新 activeAgent.value = { ...updatedAgent } // ✅ 按钮状态立即更新 } }

测试方法：

  1. 加载Agent列表
  2. 选择一个Agent
  3. 点击"启动"或"停止"
  4. 观察按钮状态是否立即更新
  5. 使用"模拟refreshAgents()"测试修复效果
