# Agent Learning Platform

一个用于学习和实践 AI Agent 开发的项目，采用前后端分离架构：
- 前端：Vue 3 + TypeScript + Vite
- 后端：FastAPI

## 文档导航

- `README.md`（本文件）：项目入口与文档索引
- `QUICK-START.md`：本地快速启动
- `PROJECT-STRUCTURE.md`：目录结构说明
- `ARCHITECTURE.md`：架构设计说明
- `PROGRESS.md`：当前进展（唯一进展主文档）
- `CONTRIBUTING.md`：贡献流程
- `API-DOCS-ACCESS.md`：API 文档访问说明
- `GITHUB-SETUP.md`：GitHub 初始化与推送说明

## 快速开始

1. 查看启动说明：`QUICK-START.md`
2. 前端开发：进入 `frontend` 后运行 `npm run dev`
3. 后端开发：进入 `backend` 后按 `backend/README.md` 说明启动服务

## 文档维护约定

- 项目入口信息统一维护在 `README.md`
- 启动相关内容仅维护在 `QUICK-START.md`
- 目录说明仅维护在 `PROJECT-STRUCTURE.md`
- 进展信息仅维护在 `PROGRESS.md`

这样可以避免重复更新和文档冲突。
