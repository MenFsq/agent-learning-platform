## 🔧 技术实现

### 前端技术栈
- **框架**: Vue 3 + TypeScript
- **UI库**: Element Plus
- **图标**: Lucide Vue
- **路由**: Vue Router
- **状态管理**: Pinia
- **构建工具**: Vite

### 后端技术栈
- **框架**: FastAPI + Uvicorn
- **文档生成**: 自动生成Swagger UI和ReDoc
- **端口**: 8001 (避免冲突)

### 数据流
```
前端 (Vue 3) → API文档页面 → 后端API (FastAPI)
      ↓              ↓              ↓
   用户界面 ← 端点列表/测试 ← 自动文档生成
```
